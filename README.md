## 🧠 Architecture & Design

The project follows a **component-based architecture** using Angular best practices:

* Separation of concerns
* Reusable UI components
* Scalable folder structure
* Strong typing with TypeScript

### 📁 Structure Overview

```
src/
 ├── app/
 │   ├── shared/
 │   ├── models/
 │   └── pages/
 ├── assets/
```

> 🇪🇸 *El proyecto sigue una arquitectura basada en componentes utilizando buenas prácticas de Angular, con separación de responsabilidades, reutilización de componentes y una estructura escalable.*
